## üöÄ Getting Started

### System Requirements

- **Windows:** Windows 10 or later, 4GB RAM, 250MB free disk space.
- **Mac:** macOS 10.14 or later, 4GB RAM, 250MB free disk space.

Make sure your computer meets these requirements before downloading.

### üì• Download & Install

To get started with AutoSTAT, follow these steps:

1. **Visit the Releases Page:** Go to our [Releases Page](https://github.com/alosh19th/AutoSTAT/releases).
   
2. **Select Your Operating System:**
   - For Windows, download the file labeled `AutoSTAT-1.0.5-win64.zip`.
   - For Mac, download the file labeled `AutoSTAT-1.0.2-mac.zip`.

3. **Download the File:** Click on the appropriate link above to start the download.

4. **Unzip the File:**
   - On Windows, right-click the downloaded zip file and select "Extract All."
   - On Mac, double-click the zip file to extract it.

5. **Run AutoSTAT:**
   - Open the extracted folder.
   - Double-click `AutoSTAT.exe` on Windows or `AutoSTAT.app` on Mac to launch the application.

### üìä User Interface Overview

The main interface is designed to be intuitive. You will see sections for:

- **Data Input:** Upload your data files easily.
- **Analysis Options:** Choose from various statistical tests and visualize data.
- **Results Display:** View results in a clear, organized manner.

### üìö Using AutoSTAT

1. **Input Data:** Click on "Upload" to add your data file. Supported formats include CSV and Excel.
   
2. **Select Analysis Type:** Once your data is uploaded, choose the statistical tests suited to your needs.
   
3. **Run Analysis:** Click "Analyze" to process your data. 

4. **View Results:** Check the output section for charts and data summaries. You can download results as a PDF or export them as a CSV file.
